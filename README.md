# Quiz Uygulaması

Bu proje, HTML, CSS ve JavaScript kullanılarak yapılmış basit bir quiz uygulamasıdır.

## Amaç

Kullanıcıya çoktan seçmeli sorular sorulur ve doğru cevaplara göre puan hesaplanır.

## Özellikler

- Çoktan seçmeli sorular
- Doğru cevap kontrolü
- Puan hesaplama
- Sonuç ekranı
- Tekrar başlatma butonu

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript

## Çalıştırma

Projeyi indirdikten sonra `index.html` dosyasını tarayıcıda açmanız yeterlidir.
